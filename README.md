# imooc-news

##### 介绍

该项目采用了uni-app框架和uniCloud云数据库实现的，主要功能模块有首页、关注、我的、设置标签、评论文章、关注作者、收藏文章

##### 使用说明

克隆到本地后使用HBuilder X 直接运行项目，即可直接运行本项目。不过数据库是云数据库，需要您自行创建一个uniCloud的数据库。初始化数据库在uniCloud-aliyun > database中的db_init.json
